# 分野の歴史

## 概要

日本十進分類法（NDC）をベースにした区分ごとに、その分野の歴史を日本語の文章でまとめるリポジトリです。区分はディレクトリ／Markdownファイルとして管理します。

> [!WARNING]
> 各項目の文章の内容は生成AIによって生成しています。
> **内容には誤りが含まれる可能性があります。**

## 項目の執筆ガイドライン

- 各区分について「分野の歴史」を叙述する（全体としてストーリーになる流れを意識）
  - NDCの区分上`2`の数字で各分野の歴史を表しているところも、文字通り「"○○史"の歴史」として、その分野における歴史認識・分野史研究などの歴史について記述する
  - 例えば「102 哲学史」の中では、"哲学"の歴史についてではなく、"哲学史"の歴史を扱う
- 目安は1項目あたり10,000文字程度
- アウトラインは階層あたり10項目以内、番号は数字で振る
- 「分野の歴史」本文ではNDC番号を出さない（見出しやファイル名は管理上の都合で例外）
- 重要な人名／書名・作品名などの固有名詞は具体名で書く
- 重要な出来事の年号は分かる範囲で書く（文脈上不要なら省略可）
- 専門用語や人名は標準的な訳語・表記を使う
- 分野の範囲は所与のものとして、その範囲がどこまでを含むか、のようなメタ視点での範囲規定の説明は入れない

## 分野の階層構造

- 原則として日本十進分類法（NDC）に準拠して分野のコードを付ける
  - ただし、NDCの分類の中で主題ではなく形式によって区分されているものは除外する
- ディレクトリの階層は、大雑把にはNDCの各桁によって親子関係を作りつつ、内容に応じて適宜グルーピング・階層のレベルを変える
  - 例えば、第一階層は0〜9の10個ではなく、以下の16個に区分する
    - **0-総記** 000-099
    - **1-哲学** 100-159
    - **16-宗教** 160-199
    - **2-歴史** 200-289
    - **29-地理．地誌．紀行** 290-299
    - **3-社会科学** 300-399
    - **4-自然科学** 400-489
    - **49-医学** 490-499
    - **5-技術．工学** 500-589
    - **59-家政学．生活科学** 590-599
    - **6-産業** 600-699
    - **7-芸術．美術** 700-779
    - **78-スポーツ．体育** 780-789
    - **79-諸芸．娯楽** 790-799
    - **8-言語** 800-899
    - **9-文学** 900-999
- ディレクトリ／ファイル名は原則として `<コード>-<名称>`で命名する
  - ただし、ディレクトリ名・ファイル名が30文字を超える場合は超えた部分を"…"で省略する
  - 階層構造のコードに範囲指定の"/"が含まれる場合は、ディレクトリ名・ファイル名では"／"に置き換える
- NDCの本表で記号が振られている範囲より細かいトピックを扱う場合は、`<コード>-<名称>：<細かいトピック>`のタイトルでファイルを作成する
  - 細分したディレクトリは作らず、ファイルは`<コード>-<名称>`のディレクトリの下に並べて置く
